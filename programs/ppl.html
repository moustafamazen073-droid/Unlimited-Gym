🔥🏋️ PUSH DAY WORKOUT PLAN

This workout plan focuses on pushing movements that target the chest, shoulders, and triceps.

Exercise Sets Reps
Incline Dumbbell Press 3 10-12
Chest Flyes 3 12-15
💪 Shoulder Press 3 8-10
Lateral Raises 3 12-15
Tricep Dips 3 10-12
Tricep Pushdowns 3 12-15

🔥 PULL DAY WORKOUT PLAN

This workout plan focuses on pulling movements that target the back, biceps, and rear delts.

Exercise Sets Reps
Pull-Ups or Lat Pulldowns 3 6-8
Barbell Rows 3 6-8
Seated Rows 3 8-10
💪 Bicep Curls 3 10-12
🔨 Hammer Curls 3 10-12
Face Pulls 3 15-20

🔥🦵 LEG DAY WORKOUT PLAN

This workout plan focuses on leg movements that target the quadriceps, hamstrings, glutes, and calves.

Exercise Sets Reps
Squats 3 6-8
Leg Press 3 8-10
Leg Extensions 3 8-10
Lunges 3 10-12 (each leg)
Leg Curls 3 10-12
Calf Raises 3 15-20
